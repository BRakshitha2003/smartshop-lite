# SmartShop Lite

A lightweight eCommerce web application built with Node.js, Express, and EJS.

## Features

- Product listing
- Product details
- Shopping cart
- Basic search
- Mock user login

## Getting Started

1. Install dependencies: `npm install`
2. Run the app: `npm start`
3. Open http://localhost:3000
